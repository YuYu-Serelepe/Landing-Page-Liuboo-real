
# LiuBoo - Inglês Divertido para Crianças (Landing Page)

## Descrição Detalhada
Este projeto consiste na Landing Page final do TCC "LiuBoo", um aplicativo educacional focado em **gamificação** para o ensino de inglês para crianças na faixa etária de **6 a 12 anos**. O site tem o objetivo principal de converter visitantes em usuários, capturando a atenção dos pais ao destacar a metodologia divertida do app e as ferramentas de segurança e controle.

---

## Objetivo do Projeto
O principal objetivo do projeto LiuBoo é criar uma solução digital que utilize a **gamificação** (Modo História, Oponentes Desafiadores) para tornar o aprendizado de inglês mais divertido e acessível para crianças. O foco pedagógico é superar a barreira da timidez e do **medo de falar**, enquanto a Área dos Pais garante **transparência** e controle sobre o processo de aprendizado e desempenho.

---

## Principais Funcionalidades
A Landing Page destaca os seguintes módulos do aplicativo:

* **Design Responsivo**: Layout otimizado para todos os tamanhos de tela (usando Bootstrap 5).
* **Gamificação em Foco**: Destaque para recursos como Modo História, Missões Diárias (Ofensiva) e Batalhas contra Oponentes (Chefões).
* **Controle Parental Avançado**: Painel dedicado para gerenciamento de perfis, controle de progresso e relatórios de desempenho.
* **Tabela de Preços Detalhada**: Comparativo claro entre o Plano Básico (Gratuito) e o Plano Premium (BooPro) nas modalidades Mensal e Anual.
* **Identidade Visual Dinâmica**: A logo do projeto muda de cor (de branco para roxo) automaticamente ao rolar a página, garantindo visibilidade em diferentes fundos.

---

## Tecnologias e Dependências
O desenvolvimento da interface utiliza um conjunto de ferramentas robustas de Front-end:

* **HTML5 e CSS3**: Base estrutural e estilização, incluindo o uso de gradientes e animações visuais.
* **Bootstrap 5**: Utilizado para a estrutura de grids, componentes (Navbar, Cards, Abas) e garantir a total responsividade do layout.
* **JavaScript/jQuery**: Responsável pela interatividade do menu, inicialização dos carrosseis (Owl Carousel) e animações (WOW.js).
* **Filtros CSS (Custom)**: Truque CSS exclusivo para mudar a cor da logo (conversão de branco para o roxo `#8E73DB`) ao rolar a página.

---

## Guia de Instalação e Customização
1.  Baixe todos os arquivos do projeto.
2.  **Personalização**: A cor roxa da logo e os detalhes dos planos de assinatura podem ser ajustados diretamente no código CSS (dentro do `<head>`) e na seção de preços.
3.  **Visualização**: Abra o arquivo `index.html` diretamente no seu navegador para rodar o projeto.
